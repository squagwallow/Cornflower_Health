# schema/

This directory will hold machine-readable schema definitions when they are ready.

Planned contents:
- `v1_fields.json` — JSON Schema or field definition file for the v1 internal schema
- `notion_properties.json` — Notion database property configuration (property names and types only; no IDs or tokens)

Until these files exist, the canonical schema is defined in [`docs/schema-plan.md`](../docs/schema-plan.md).
